Introdução ao HTML

Principais Tags

Do inglês HTML significa Hyper Text Markup Language. Esta é a linguaguem padrão de marcação para criação de página Web. Usa-se tags para fazer as marcações.

As tags são representadas pelos sinais de menor e maior e dentro destes sinais o nome da tag por ex: <nome_da_tag> e a maioria das tags são abertas e fechadas, apresentando o sinal de barra (/) dentro da tags, no caso, nas tags de fechamentos. ex: </nome_da_tag>

Tag Função
<html> Define o início e o fim de um documento HTML.
<head> Contém meta-informações sobre o documento (ex.: título, charset, estilos).
<title> Define o título do documento que aparece na aba do navegador.
<meta> Define metadados sobre o documento, como o charset e descrições.
<link> Utilizada para vincular arquivos externos, como folhas de estilo CSS.
<style> Contém regras de estilo CSS internas.
<body> Define o corpo do documento, onde o conteúdo visível é colocado.
<header> Representa o cabeçalho de uma seção ou página.
<nav> Define uma seção de navegação, contendo links para outras páginas ou partes do mesmo documento.
<main> Contém o conteúdo principal do documento.
<section> Define uma seção de um documento, como um capítulo ou uma parte temática.
<article> Representa um conteúdo independente, como um artigo de jornal ou blog post.
<aside> Define um conteúdo secundário ou relacionado, como uma barra lateral.
<footer> Representa o rodapé de uma seção ou página.
<h1> a <h6> Define os títulos e subtítulos, com <h1> sendo o mais importante e <h6> o menos importante.
<p> Define um parágrafo de texto.
<a> Define um hyperlink para outra página ou seção do documento.
<img> Insere uma imagem no documento.
<ul> Define uma lista não ordenada (com marcadores).
<ol> Define uma lista ordenada (numerada).
<li> Define um item em uma lista.
<table> Define uma tabela.
<tr> Define uma linha de tabela.
<td> Define uma célula de tabela.
<th> Define uma célula de cabeçalho de tabela.
<form> Define um formulário para entrada de dados do usuário.
<input> Define um campo de entrada em um formulário.
<button> Define um botão clicável.
<div> Define uma divisão ou seção genérica no documento.
<span> Define uma seção genérica em linha para estilização.
<strong> Define texto importante, geralmente exibido em negrito.
<em> Define texto enfatizado, geralmente exibido em itálico.
<br> Insere uma quebra de linha.
<hr> Insere uma linha horizontal para separação de conteúdo.
<blockquote> Define uma citação em bloco.
<code> Define um trecho de código fonte.
<pre> Define texto pré-formatado.
